
RUNNING THE PROJECT

1. Install Node.js
2. Open terminal inside project folder
3. Run:

npm install
npm run dev

4. Open the shown localhost URL (usually http://localhost:5173)

PAGES
/ -> Dashboard
/local-users -> Local JSON users
/users-api -> JSONPlaceholder users
/fake-posts -> DummyJSON posts using Axios

FEATURES
- Fetch local JSON
- Fetch public API
- Axios API integration
- Refresh button
- Dropdown filter
- React Router navigation
- Basic CSS UI
